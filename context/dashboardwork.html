Dashboard Work Context

Implementation notes for the protected Mentoria Hub dashboard, courses, lessons, and MentorPet lesson notes.

Centered Minimalist Page Rule

All protected dashboard and workspace pages must use a centered minimalist layout: constrained center-aligned shells, calm spacing, no sidebars, no left or right page drift, and restrained glass surfaces that match the Mentoria dark ethnofuturist design system.

Routing

The protected workspace uses browser routes for dashboard, courses, course detail, lesson detail, opportunity search, and MentorPet. Protected routes should show the auth flow when no student profile is available.

  • /dashboard - student dashboard and overview.
  • /courses - live course catalog sourced from data.
  • /courses/:courseId - course detail and lesson list.
  • /courses/:courseId/lessons/:lessonId - lesson page with video and support material.
  • /opportunities - opportunity search workspace.
  • /mentor-pet - future MentorPet notes workspace.

Course And Lesson Data

Courses and lessons remain static data for now, but the shape is admin-ready. Course additions should automatically appear on the courses page. Lesson video URLs are optional and intended to be admin-managed later.

MentorPet Notes

MentorPet notes are a separate future function, not normal lessons. Notes attach to a specific courseId and lessonId, appear as smaller right-aligned items below a lesson in course lists, and appear as expandable items at the bottom of the lesson page.
